## How to start the SIEM project

**Step 1: Put your log files in the logs folder**
- Copy your downloaded `.log` files into logs
- Example: `logs/auth.log`, `logs/access.log`, etc.

**Step 2: Analyze the logs (one-time)**
```bash
python main.py --batch --log-file logs
```
This will:
- Find all `.log` files in logs
- Parse them
- Detect suspicious activity
- Save results to `siem.db`

**Step 3: View the dashboard**
```bash
python main.py --serve
```
Then open your browser to:
```
http://127.0.0.1:5000
```

You'll see:
- Recent alerts table
- Alerts by severity (pie chart)
- Top offending IPs (bar chart)
- Alert timeline

---

**Optional: Monitor logs in real-time**
```bash
python main.py --watch --log-file logs
```
This watches for new/modified `.log` files and alerts you as suspicious activity is detected.

That's it! Start with step 1-3 to get results on the dashboard.
